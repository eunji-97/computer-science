# 데이터베이스란
데이터베이스(Database, DB)
구조화된 정보를 저장하고 관리하기 위한 데이터 집합, 저장소를 의미한다.

## 종류
관계형 데이터베이스 : 행-열로 구성된 테이블 형태로 저장. SQL을 사용한다. 
비관계형 데이터베이스 : 키-값, 문서, 그래프 등의 다양한 형태로 저장
백터 데이터베이스 : 비정형 데이터를 벡터 임베딩(숫자 벡터) 형태로 저장. 벡터 간 유사성 검색을 수행하는 데이터베이스

## 구성 요소
사용자 : 데이터를 활용하는 사람
응용 프로그램 : 데이터를 활용하는 소프트웨어
DBMS : 데이터베이스 관리 시스템 (생성, 저장, 관리, 조회할 수 있도록 기능을 제공하는 소프트웨어. Postgresql, MongoDB, Redis 와 같은 것이 예)
데이터베이스 : 실제 데이터의 저장되는 공간
