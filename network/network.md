# 네트워크란
네트워크(Network)
여러 컴퓨터, 장치가 서로 데이터를 주고 받을 수 있도록 통신 링크로 연결된 구조를 의미이다. 망이라고도 한다.

어떻게 연결되는지
이 장비들은 서로 통신하기 위해 물리적인 매체로 연결되어야 하는데, 구리 케이블 및 광섬유와 같은 유선 매체, 무선 주파수(wi-fi), 이동통신(LTE, 5G)와 같은 무선 매체가 있다.

- 프로토콜
네트워크를 통해 통신하기 위한 합의된 통신 규칙, 절차를 의미한다.
ex) TCP, IP, HTTP, FTP

- 사용 예시
웹 엑세스, 서버 공유, 프린터와 팩스 기기, 메시징 서비스
